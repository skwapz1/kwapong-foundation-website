# Mobile App Screenshots

Place the following screenshots from the Crimson Warriors mobile app in this directory:

## Required Screenshots:

### 1. home-screen.png
- Screenshot of the daily check-in / home screen
- Should show the wellness indicator or daily health tracking
- Recommended size: 400x800px (portrait orientation)

### 2. medications-screen.png
- Screenshot of the medications tracking screen
- Should show the medication list with checkmarks/reminders
- Recommended size: 400x800px (portrait orientation)

### 3. rewards-screen.png
- Screenshot of the achievements/rewards screen
- Should show trophies, points, or gamification elements
- Recommended size: 400x800px (portrait orientation)

## How to Capture Screenshots:

### For iOS Simulator:
1. Open the Crimson Warriors app in the iOS simulator
2. Navigate to each screen mentioned above
3. Press `Cmd + S` to save a screenshot
4. Screenshots will be saved to your Desktop
5. Rename and move them to this directory

### For Android Emulator:
1. Open the Crimson Warriors app in the Android emulator
2. Navigate to each screen mentioned above
3. Click the camera icon in the emulator toolbar
4. Screenshots will be saved to your Desktop
5. Rename and move them to this directory

### For Real Device:
1. Open the Crimson Warriors app on your device
2. Navigate to each screen
3. Take a screenshot using device controls
4. Transfer to your computer and place in this directory

## Image Optimization:
- Export as PNG for best quality
- Recommended dimensions: 400x800px or similar portrait ratio
- Optimize file size if needed using tools like TinyPNG
